# Search Optimizer

A modern Google Search cleanup extension.

## Features
- Hide entire Google search modules (Sponsored, Videos, Latest Posts, Products, People Also Ask) via live popup controls.
- Permanently banish specific websites from ever showing up in your Google search results again.
- Pure Google compatibility: No network request blocking, just pure cosmetic DOM filtering.

## Installation
1. Download code and unzip.
2. Go to `chrome://extensions/` in Chromium-based browsers.
3. Enable "Developer mode".
4. Click "Load unpacked" and select this repository folder.
